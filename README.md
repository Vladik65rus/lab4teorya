## 📝 Постановка задачи

Разработать модуль поиска подстрок с использованием регулярных выражений, интегрировать его в существующее приложение (текстовый редактор) и обеспечить наглядный вывод результатов.

### Требования к программе:

1. Интегрировать модуль поиска в существующий интерфейс текстового редактора (ЛР1)
2. Добавить в интерфейс элементы управления для выбора типа поиска (выпадающий список или кнопки) и запуска анализа
3. Результаты поиска должны отображаться в таблице с колонками:
   - Найденная подстрока (извлеченный фрагмент)
   - Начальная позиция (номер строки, номер символа)
   - Длина (количество символов)
4. При выборе строки в таблице результатов соответствующая подстрока в области редактирования должна подсвечиваться (изменение цвета фона или выделение)
5. Возможность вывода количества найденных совпадений

### Входные и выходные данные:

- **Вход:** текст, введенный пользователем в область редактирования
- **Выход:** таблица найденных подстрок с указанием позиций, длины и количества совпадений; подсветка соответствующих фрагментов в исходном тексте

## 🔍 Решение задач

### Задача 1: Поиск HEX-цветов (формат #RGB)

#### Описание задачи

Необходимо найти все вхождения HEX-кодов цветов в формате `#RGB`, где `R`, `G`, `B` — шестнадцатеричные цифры (0-9, A-F, a-f). Пример: `#F00` (красный), `#0F0` (зеленый), `#00F` (синий), `#FFF` (белый), `#000` (черный).

#### Регулярное выражение

```regex
#[0-9A-Fa-f]{3}\b
```
Задача 2: Поиск идентификаторов в языке Rust
Описание задачи
##Необходимо найти все идентификаторы в языке Rust. Идентификатор должен начинаться с буквы (a-z, A-Z), символа подчеркивания _ или знака доллара $, за которыми могут следовать буквы, цифры, символы подчеркивания.

## Регулярное выражение
[a-zA-Z$_][a-zA-Z0-9$_]*\b
### Пояснение каждого обозначения:
Символ	Значение
[a-zA-Z$_]	Первый символ: буква (a-z или A-Z), $ или _
[a-zA-Z0-9$_]*	Последующие символы (0 или более): буквы, цифры, $, _
\b	Граница слова (чтобы не находить части слов)
Примеры строк, которые должны находиться:
Строка	Найденные подстроки
let x = 5;	x
fn calculate_sum()	calculate_sum
$_private	$_private
myVar123	myVar123
_underscore	_underscore
$special	$special
Примеры строк, которые НЕ должны находиться:
Строка	Почему не находится
123abc	Начинается с цифры
@invalid	Начинается с @
my-var	Содержит дефис (недопустимый символ)
fn	Ключевое слово (должно быть исключено в лексическом анализаторе)
Задача 3: Поиск дат в формате MM/DD/YYYY
Описание задачи
Необходимо найти все даты в формате MM/DD/YYYY с валидацией:

Месяц: 01-12

День: от 01 до 31 (с учетом количества дней в месяце)

Год: 1900-2099

Учет високосных годов (февраль имеет 29 дней)

Регулярное выражение
```
(?:(?:0[1-9]|1[0-2])/(?:0[1-9]|[12][0-9]|3[01])/(?:19|20)\d{2})
```
##Пояснение каждого обозначения:
### Символ	Значение
(?: ... )	Незахватывающая группа (группировка без сохранения)
(?:0[1-9]|1[0-2])	Месяц: 01-09 или 10-12
/	Разделитель (слеш)
(?:0[1-9]|[12][0-9]|3[01])	День: 01-09, 10-29 или 30-31
/	Разделитель (слеш)
(?:19|20)	Год: 19xx или 20xx
\d{2}	Две любые цифры (xx)
Примеры строк, которые должны находиться:
Строка	Найденные подстроки	Примечание
01/15/2024	01/15/2024	Корректная дата
12/31/2020	12/31/2020	Последний день года
02/29/2020	02/29/2020	Високосный год (2020)
04/30/2023	04/30/2023	Апрель имеет 30 дней
06/01/1999	06/01/1999	1999 год
Примеры строк, которые НЕ должны находиться:
Строка	Почему не находится
13/01/2024	Месяц 13 (недопустим)
02/30/2023	Февраль не имеет 30 дней
04/31/2023	Апрель имеет только 30 дней
02/29/2023	2023 не високосный год
01-15-2024	Используется дефис вместо слеша
01/15/24	Год только из 2 цифр
Дополнительная валидация в коде:
После нахождения подстроки по регулярному выражению, выполняется дополнительная проверка на корректность даты с учетом:

### Количества дней в месяце

```
def _validate_date(self, date_str):
    """Проверка корректности даты с учетом високосных годов"""
    try:
        month, day, year = map(int, date_str.split('/'))
        
        # Проверка месяца
        if month < 1 or month > 12:
            return False
        
        # Количество дней в месяце
        days_in_month = [31, 29 if (year % 4 == 0 and year % 100 != 0) or (year % 400 == 0) else 28,
                        31, 30, 31, 30, 31, 31, 30, 31, 30, 31]
        
        # Проверка дня
        if day < 1 or day > days_in_month[month - 1]:
            return False
        
        return True
    except:
        return False
        ```
## 💻 Реализация в программе
### Класс RegexSearcher

```
class RegexSearcher:
    """Класс для поиска подстрок с помощью регулярных выражений"""
    
    def __init__(self):
        self.results = []
    
    def search_hex_color(self, text):
        """Поиск HEX-цветов в формате #RGB"""
        pattern = r'#[0-9A-Fa-f]{3}\b'
        return self._search_pattern(text, pattern, "HEX цвет (3 символа)")
    
    def search_identifier(self, text):
        """Поиск идентификаторов в Rust"""
        pattern = r'[a-zA-Z$_][a-zA-Z0-9]*\b'
        return self._search_pattern(text, pattern, "Идентификатор")
    
    def search_date(self, text):
        """Поиск дат в формате MM/DD/YYYY с валидацией"""
        pattern = r'(?:(?:0[1-9]|1[0-2])/(?:0[1-9]|[12][0-9]|3[01])/(?:19|20)\d{2})'
        matches = self._search_pattern(text, pattern, "Дата MM/DD/YYYY")
        
        # Дополнительная валидация дат
        valid_matches = []
        for match in matches:
            if self._validate_date(match['match']):
                valid_matches.append(match)
            else:
                match['type'] = "❌ Некорректная дата"
                valid_matches.append(match)
        
        return valid_matches
    
    def _search_pattern(self, text, pattern, type_name):
        """Общий метод поиска по регулярному выражению"""
        results = []
        compiled = re.compile(pattern, re.IGNORECASE)
        
        for match in compiled.finditer(text):
            start = match.start()
            end = match.end()
            matched_text = match.group()
            
            # Вычисление номера строки и позиции
            line_num = text[:start].count('\n') + 1
            last_newline = text[:start].rfind('\n')
            col_num = start - last_newline if last_newline != -1 else start + 1
            
            results.append({
                'type': type_name,
                'match': matched_text,
                'line': line_num,
                'col': col_num,
                'start': start,
                'end': end
            })
        
        return results
    
    def _validate_date(self, date_str):
        """Проверка корректности даты с учетом високосных годов"""
        try:
            month, day, year = map(int, date_str.split('/'))
            
            if month < 1 or month > 12:
                return False
            
            days_in_month = [31, 
                            29 if (year % 4 == 0 and year % 100 != 0) or (year % 400 == 0) else 28,
                            31, 30, 31, 30, 31, 31, 30, 31, 30, 31]
            
            if day < 1 or day > days_in_month[month - 1]:
                return False
            
            return True
        except:
            return False
            ```
